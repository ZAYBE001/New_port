## PORTFOLIO
* A dynamic, high-performance personal portfolio website built with **React**, **Vite**, and **Tailwind CSS**. This interactive application showcase is fully responsive, optimized for modern displays, and integrated with live API configurations to deliver a seamless user experience.
## 🚀 Live Demo

🔗 **Explore Live:** [https://zaybe001.github.io/]

---

## ✨ Features

*   **Dynamic Project Ingestion:** Integrates seamlessly with the **GitHub REST API** to automatically fetch and display repository assets, update counts, and build statuses in real time.
*   **Modern Interactive UI/UX:** Features fluid animations, modular components, and a fully responsive grid system styled with utility-first **Tailwind CSS**.
*   **Fully Managed Contact Pipeline:** Powered by **Web3Forms API** ingestion for frictionless, client-side email delivery directly to my inbox without requiring a custom backend server.

## 🛠️ Tech Stack

*   **Frontend Library:** React 18+ (Hooks-driven state management)
*   **Build Tooling:** Vite
*   **Styling & Themes:** Tailwind CSS
*   **APIs & Services:** GitHub REST API, Web3Forms
*   **Hosting & Deployment:** GitHub Pages
